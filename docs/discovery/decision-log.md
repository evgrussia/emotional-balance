### DEC-001: Telegram WebApp как основной клиент

| Поле | Значение |
|------|----------|
| **ID** | DEC-001 |
| **Дата** | 2026-02-04 |
| **Статус** | Утверждено |
| **Фаза** | Intake |
| **Агент** | Orchestrator |
| **Stakeholder** | Пользователь |

**Решение:** Telegram WebApp — единственный клиент в MVP.

**Обоснование:**
- Мгновенный доступ в момент тревоги (не нужно скачивать приложение)
- Низкий барьер входа — пользователи уже в Telegram
- Push-уведомления через Telegram Bot API
- Нулевой CAC для привлечения из Telegram-каналов

**Альтернативы рассмотрены:**
- Нативные приложения (iOS/Android) — высокая стоимость, длинный цикл разработки
- PWA (Progressive Web App) — ограниченные push-уведомления
- Мультиплатформа (Telegram + Web) — размытие фокуса на MVP

**Риски:**
- Зависимость от Telegram API и политики
- Ограничения WebApp SDK
- Митигация: абстракция UI, план миграции на standalone

| **Статус** | Утверждено |
| **Фаза** | Discovery (дополнение) |
| **Агент** | Product / Research |

**Решение:** Добавить терапевтическую геймификацию (FR-018) как P1 фичу с guilt-free подходом.

**Обоснование:**
- Исследования показывают retention +40-60% при терапевтической геймификации
- Ключевое отличие от обычной геймификации: нет наказания за пропуск, все достижения имеют терапевтический смысл
- Визуальная метафора «дерево здоровья» — уникальна, создаёт эмоциональную привязку
- Ни один российский конкурент не использует терапевтическую геймификацию

**Риски:**
- Геймификация может trivialize терапию — митигация: каждое достижение привязано к реальному терапевтическому действию
- Guilt-free подход: «Пауза — это тоже забота о себе» — нет агрессивного сброса серий

### DEC-036: Paywall после 3-5 AI-сессий (оптимальный момент конверсии)

| Поле | Значение |
|------|----------|
| **ID** | DEC-036 |
| **Дата** | 2026-02-04 |
| **Статус** | Утверждено |
| **Фаза** | Discovery |
| **Агент** | Business-Analyst |
| **Stakeholder** | Orchestrator |

**Решение:** Показывать paywall после 3-5 сессий с AI, когда пользователь уже получил value и готов платить.

**Обоснование:**

- SOS/Crisis ВСЕГДА бесплатно (DEC-027) — не блокируются paywall-ом

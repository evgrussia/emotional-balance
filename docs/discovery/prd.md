# PRD: Emotional Balance (Нейро-Психолог 24/7) — MVP

## 1. Executive Summary

**Emotional Balance** — ИИ-платформа психологической поддержки, доступная 24/7 через Telegram WebApp. Платформа использует российские LLM (GigaChat/YandexGPT) и методы когнитивно-поведенческой терапии (КПТ) для помощи пользователям в управлении тревожностью, стрессом и эмоциональным состоянием.

MVP включает полный набор функций: ИИ-чат, голосовые сессии, дневник эмоций, КПТ-упражнения, детектор кризисных состояний, маркетплейс психологов, медитации, интеграция с wearables, B2B-управление, SOS-протоколы, терапевтический мост, мини-курсы, реферальная программа и геймификация.

**Целевой рынок:** 15M+ россиян с тревожностью/депрессией.
**Монетизация:** Freemium + подписка 990-2990 руб/мес + B2B wellness.

## 2. Problem Statement

### 2.1 Проблема пользователя

Люди, испытывающие тревожность, стресс и депрессию, не могут получить доступную, мгновенную и квалифицированную психологическую поддержку в момент, когда она необходима.

### 2.2 Причины

| Барьер | Описание | Влияние |
|--------|----------|---------|
| **Финансовый** | Сессия у психолога: 3 000-7 000 руб | Недоступно для 70%+ населения |
| **Временной** | Запись за дни/недели, сессия 50 мин | Помощь недоступна в момент кризиса |
| **Географический** | 5 психологов на 100 000 в регионах | Физически недоступно |
| **Стигматизация** | 60% не обращаются из-за страха | Проблема не решается |
| **Информационный** | Не знают с чего начать | Высокий порог входа |

### 2.3 Существующие альтернативы и их недостатки

- **Живые психологи:** Дорого, долго, ограниченная доступность
- **Zigmund/YouTalk:** Только живые специалисты, высокая цена, нет 24/7
- **Replika/Woebot:** Не на русском, не соответствуют ФЗ-152, нет КПТ-протоколов
- **Медитационные приложения:** Не персонализированы, нет терапевтического подхода

## 10. Risks & Mitigations

| # | Риск | Вероятность | Влияние | Мера |

| R-10 | Перегрузка бюджета на LLM API | Средняя | Высокое | Настраиваемые лимиты (FR-013), алерты, fallback-стратегии |

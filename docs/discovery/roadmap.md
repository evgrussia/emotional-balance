# Product Roadmap: Emotional Balance

## 1. Обзор

### 1.1 Стратегия релизов

```
Phase 0: Foundation    → Архитектура, инфраструктура, core
Phase 1: MVP Alpha     → Основные фичи, закрытый тест
Phase 2: MVP Beta      → Все P0, открытый бета-тест
Phase 3: Public Launch → Полный запуск
Phase 4: Growth        → P1 фичи, масштабирование
Phase 5: Scale         → P2 фичи, новые рынки
```

### 1.2 Принципы приоритизации

1. **Safety first** — детектор кризисов и безопасность пользователей
2. **Core value** — ИИ-чат и КПТ-упражнения
3. **Engagement loop** — дневник + push + привычка
4. **Monetization** — подписки + маркетплейс
5. **Expansion** — B2B + wearables + голос

## 9. Риски по фазам

| Фаза | Риск | Митигация |
|------|------|-----------|
| Foundation | LLM API нестабилен | Поддержка 2 LLM, fallback |
| Alpha | Качество КПТ-диалогов низкое | RAG + итерации промптов + экспертный аудит |
| Alpha | Детектор кризисов: false negatives | Консервативный порог, экспертная разметка |
| Beta | STT/TTS качество на русском | Тестирование нескольких моделей (Whisper, XTTS) |
| Beta | Психологи не регистрируются | 0% комиссия, прямой onboarding |
| Beta | YooKassa интеграция сложнее ожидаемого | Ранний прототип оплаты |
| Launch | Низкий adoption | Freemium + контент-маркетинг + referral |
| Growth | Churn выше 8% | Engagement-петли, персонализация, push |

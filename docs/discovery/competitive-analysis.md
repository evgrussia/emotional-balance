## 5. Market Gaps (возможности)

### Gap 1: Отсутствие полноценного ИИ-психолога на российских LLM
**Описание:** Существующие российские ИИ-боты (Ася, Лея, Sabina) используют GPT или базовые модели без структурированных КПТ-протоколов. Нет продукта на GigaChat/YandexGPT с верифицированной базой КПТ.
**Наша возможность:** Первый ИИ-психолог на российских LLM с RAG на верифицированных КПТ-протоколах.

### Gap 2: Нет гибрида ИИ + маркетплейс живых специалистов
**Описание:** Рынок разделён: либо платформы живых специалистов (Ясно, Zigmund, YouTalk), либо ИИ-боты (Ася, Лея). Никто не объединяет оба подхода в единую экосистему.
**Наша возможность:** Встроенный маркетплейс с ИИ-рекомендацией специалиста на основе данных пользователя.

### Gap 3: Нет wearables-интеграции для mental health в РФ
**Описание:** Ни один российский сервис не интегрируется с Apple Watch или Mi Band для мониторинга стресса.
**Наша возможность:** Первая интеграция wearables + дневник эмоций + ИИ-анализ в РФ.

### Gap 4: Дневник эмоций с ИИ-аналитикой отсутствует
**Описание:** Ни у одного российского конкурента нет дневника эмоций с автоматическим ИИ-анализом паттернов, триггеров и корреляций.
**Наша возможность:** Дневник + ИИ-инсайты + wearables-данные = уникальное ценностное предложение.

### Gap 5: Голосовые ИИ-сессии на русском языке
**Описание:** Ни один российский ИИ-бот не предлагает полноценные голосовые сессии (STT/TTS). Replika делает это на английском.
**Наша возможность:** Self-hosted STT/TTS для голосовых ИИ-сессий на русском.

### Gap 6: Автоматический детектор кризисов в российских ИИ-ботах
**Описание:** Российские ИИ-боты не имеют автоматического детектора суицидальных намерений с эскалацией.
**Наша возможность:** Safety-critical фича, отличающая нас от конкурентов и создающая доверие.

### Gap 7: SOS-протоколы мгновенной помощи
**Описание:** Ни один конкурент не предлагает моментальный доступ к пошаговым протоколам для острых состояний (паника, бессонница, тревога). Wysa имеет базовые SOS-элементы, но не на русском и не в Telegram. Российские ИИ-боты не имеют SOS-функционала.
**Наша возможность:** SOS-кнопка в 1 тап с 6 ситуативными протоколами, голосовым сопровождением и автоматической записью в дневник. Бесплатно для всех тарифов.
**Тренд:** EMI (Ecological Momentary Intervention) — один из главных трендов mental health 2025-2026.

### Gap 8: Терапевтический мост между ИИ и живым специалистом
**Описание:** Рынок разделён на ИИ-ботов и платформы живых специалистов. Нигде нет бесшовного перехода с передачей контекста. Клиент начинает рассказ заново каждому специалисту.
**Наша возможность:** ИИ-саммари из чата, дневника и упражнений → живой психолог получает контекст до сессии. Обратная связь психолога улучшает ИИ. Уникальный дифференциатор маркетплейса.

### Gap 9: Образовательный контент в российских mental health приложениях
**Описание:** Российские конкуренты не предлагают структурированных образовательных мини-курсов. Пользователи вынуждены искать информацию в разрозненных источниках (YouTube, статьи, книги).
**Наша возможность:** Мини-курсы 5-10 мин с тестами и практикой. Контент на основе верифицированных КПТ-протоколов. Engagement через серии и прогресс.

### Gap 10: Терапевтическая геймификация
**Описание:** Replika имеет базовую геймификацию, но без терапевтического смысла. Российские конкуренты не используют геймификацию. Исследования показывают, что терапевтическая геймификация повышает retention на 40-60%.
**Наша возможность:** «Дерево эмоционального здоровья» — визуальная метафора роста с привязкой к терапевтическим активностям. Guilt-free подход (нет наказания за пропуск).

### Gap 11: Виральный рост через Telegram
**Описание:** Ни один российский mental health сервис не использует реферальную программу через Telegram deep links. Стандартный CAC в отрасли — 500-3000 руб/пользователь.
**Наша возможность:** Реферальная программа с бонусом Premium для обоих, CAC реферальных ~0 руб. Telegram deep links для мгновенной регистрации. При 10% referral rate — существенное снижение blended CAC.

- Полное соответствие ФЗ-152
- Структурированные КПТ-протоколы (не фристайл-чат)

- **SOS-протоколы мгновенной помощи** (EMI-тренд, 6 ситуаций, 1 тап)
- **Терапевтический мост ИИ↔Психолог** (уникальный дифференциатор, никто не делает)
- **Терапевтическая геймификация** (guilt-free, научно обоснованная, retention +40-60%)
- **Реферальная программа через Telegram** (виральный рост, CAC рефералов ~0)

### Weaknesses (Слабые стороны)
- Новый продукт без базы пользователей и social proof
- Зависимость от качества GigaChat/YandexGPT (возможные галлюцинации)
- Self-hosted STT/TTS — высокие затраты на GPU
- Широкий MVP scope — риск размытого фокуса
- Нет клинической валидации (RCT) на старте

### Opportunities (Возможности)
- Рынок растёт 8-15% CAGR, особенно онлайн-сегмент
- ИИ mental health «ещё очень сырой» в России — окно для first mover
- B2B wellness-сегмент активно растёт
- Закрытие Woebot D2C показывает потребность в новых решениях
- Регулирование (реестры, стандарты) может стать барьером для мелких конкурентов

### Threats (Угрозы)
- Ясно/Zigmund могут запустить собственных ИИ-ассистентов
- Регулирование ИИ в healthcare может ограничить продукт
- GigaChat/YandexGPT могут изменить ценовую политику или API
- Общественное недоверие к ИИ-психологам
- Telegram может изменить политику WebApp

## 7. Рекомендации

### 7.1 Стратегические

1. **Позиционирование:** «Первый ИИ-психолог на российских LLM с научно обоснованными КПТ-методиками и возможностью записи к живому специалисту» — занять пустую нишу гибрида.

2. **Дифференциация через данные:** Дневник эмоций + wearables + ИИ-анализ = персонализация, которую не может предложить ни один конкурент.

3. **Safety как конкурентное преимущество:** Детектор кризисов + этические принципы + disclaimers → доверие и PR-потенциал.

4. **Партнёрства с психологами:** Привлечь специалистов с Ясно/YouTalk, предложив дополнительный канал клиентов через ИИ-рекомендации.

### 7.2 Тактические

5. **Telegram-first — быстрый старт:** Нулевой барьер входа через Telegram даёт преимущество перед приложениями.

6. **Freemium для базы:** Бесплатный тариф с ограничениями для накопления пользовательской базы и social proof.

7. **B2B через кейсы:** Первые 5-10 корпоративных клиентов по сниженной цене → кейсы → масштабирование.

8. **Контент-маркетинг:** Партнёрства с психологами-блогерами в Telegram для органического роста.

## 2. Сегменты рынка

### 2.1 Карта сегментов

| Сегмент | Размер | Рост | Наша релевантность |
|---------|--------|------|-------------------|
| **Онлайн-психотерапия (живые)** | 15-20 млрд руб | 15-20% | Частичная (маркетплейс) |
| **Mental health apps (ИИ)** | 1-3 млрд руб | 25-40% | Прямая (core продукт) |
| **Корпоративный wellness** | 5-8 млрд руб | 20-30% | Прямая (B2B) |
| **Медитационные приложения** | 2-4 млрд руб | 10-15% | Частичная (медитации) |
| **Wearables health** | 10-15 млрд руб | 15-20% | Дополнительная |

### 2.2 Целевые сегменты (приоритет)

**Сегмент 1: B2C — Городские миллениалы с тревожностью (Primary)**
- Возраст: 25-35 лет
- Города: Москва, СПб, миллионники
- Доход: 80-200K руб/мес
- Характеристики: технически продвинуты, используют Telegram, готовы к ИИ
- Размер: ~5M человек
- Потенциальная конверсия: 3-5%

**Сегмент 2: B2C — Жители регионов с ограниченным доступом**
- Возраст: 25-45 лет
- Города: <500K населения
- Доход: 40-80K руб/мес
- Характеристики: нет доступа к психологам, ценочувствительны
- Размер: ~8M человек
- Потенциальная конверсия: 2-3%

**Сегмент 3: B2B — IT-компании и крупный бизнес**
- Размер компании: 50-5000 сотрудников
- Индустрии: IT, финтех, консалтинг, телеком
- Бюджет wellness: 1000-3000 руб/сотрудник/мес
- Размер: ~10 000 компаний
- Потенциальная конверсия: 0.5-1%

**Сегмент 4: B2C — Профессионалы с выгоранием**
- Возраст: 30-45 лет
- Профессии: менеджеры, предприниматели, IT
- Доход: 150-500K руб/мес
- Характеристики: мало времени, ценят эффективность, готовы платить
- Размер: ~2M человек
- Потенциальная конверсия: 5-8%

## 5. Барьеры входа

### 5.1 Для нашего продукта

| Барьер | Уровень | Наше решение |
|--------|---------|-------------|
| **Доступ к LLM API** | Низкий | GigaChat/YandexGPT — открытые API |
| **Технологическая сложность** | Средний | LangChain/LangGraph, self-hosted STT/TTS |
| **Контент (КПТ-база)** | Средний | RAG на верифицированных протоколах |
| **Привлечение психологов** | Высокий | Онбординг с 0% комиссией, агрессивный маркетинг |
| **Пользовательская база** | Высокий | Freemium + Telegram (нулевой барьер) |
| **Регуляторные требования** | Средний | ФЗ-152 compliance, российские LLM, данные в РФ |
| **Клиническая валидация** | Высокий | Партнёрства с университетами (план) |
| **Финансирование GPU (STT/TTS)** | Средний | Оптимизация моделей, настраиваемые лимиты |

### 5.2 Для конкурентов (наши преимущества как барьеры)

| Барьер для конкурентов | Описание |
|----------------------|----------|
| **First mover в гибриде ИИ + маркетплейс (РФ)** | Сетевой эффект: чем больше пользователей, тем больше психологов и наоборот |
| **Данные и персонализация** | Накопленные данные дневников + wearables + чатов = лучшая персонализация |
| **КПТ-база знаний** | Верифицированная RAG-база — требует экспертизы и времени |
| **Telegram-интеграция** | Глубокая интеграция с WebApp, ботом, push-уведомлениями |

## 8. Резюме

### Ключевые выводы

1. **Рынок большой и растущий** — TAM ~50 млрд руб к 2027, CAGR 8-15%
2. **Ниша ИИ mental health в РФ практически пуста** — окно для first mover
3. **B2B wellness — самый быстрорастущий сегмент** — 20-30% CAGR
4. **Telegram-first — уникальная стратегия** — нулевой барьер входа
5. **Гибрид ИИ + маркетплейс — незанятая позиция** — ни один конкурент не объединяет оба подхода
6. **Регуляторные тренды в нашу пользу** — ФЗ-152, российские LLM как преимущество
7. **SOM реалистичный** — 22M руб в Год 1, 690M руб в Год 3

### Рекомендации для стратегии

- **Фокус на Сегмент 1 и 4** — городские миллениалы и профессионалы с выгоранием (высокая платёжеспособность)
- **Быстрый запуск B2B пилотов** — 5-10 IT-компаний в первые 3 месяца
- **Инвестиции в контент-маркетинг** — Telegram-каналы, партнёрства с психологами
- **Клиническая валидация как долгосрочное преимущество** — начать планирование RCT

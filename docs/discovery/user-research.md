## 2. Персоны (детализированные)

### Persona 1: Аня — «Тревожная миллениалка»

| Атрибут | Значение |
|---------|----------|
| **Возраст** | 28 лет |
| **Город** | Москва |
| **Профессия** | Маркетолог в IT-компании |
| **Доход** | 120 000 руб/мес |
| **Образование** | Высшее (журналистика) |
| **Семейное положение** | Не замужем, живёт одна |
| **Telegram-активность** | 3+ часа/день, 15+ каналов |
| **Техническая грамотность** | Высокая |
| **Опыт с психологами** | Ходила 3 раза, бросила (дорого + стигма) |

**Дневная рутина:**
- 8:00 — просыпается, проверяет Telegram
- 9:00-18:00 — работа (удалённо/гибрид), высокий стресс
- 19:00-22:00 — соцсети, сериалы, звонки подругам
- 23:00-01:00 — бессонница, тревожные мысли, поиск информации о тревожности
- **Кризисный момент:** 2:00 ночи — паническая атака, некому позвонить

**Мотивации:**
- Хочет управлять тревогой без таблеток
- Ищет что-то между «ничего не делать» и «ходить к психологу»
- Ценит анонимность и удобство
- Готова платить, если видит результат

**Фрустрации:**
- «Психолог стоит дорого, а эффект не сразу»
- «Медитации все одинаковые и не персонализированы»
- «Ночью помощь недоступна»
- «Не хочу рассказывать знакомым о проблемах»

**Тариф:** Standard (990 руб/мес) — после 7-дневного trial

| **Опыт с психологами** | Никогда не обращался |

**Дневная рутина:**
- 7:00-8:00 — утро с семьёй, спешка
- 8:30-19:00 — работа: совещания, конфликты, дедлайны
- 19:30-21:00 — семья, ужин

**Фрустрации:**

- «Раздражительность портит отношения с семьёй»
- «Медитации — это ерунда» (скептик, но открыт к КПТ)

### Persona 3: Марина — «HR-директор»

| Атрибут | Значение |
|---------|----------|
| **Возраст** | 40 лет |
| **Город** | Санкт-Петербург |
| **Профессия** | HR-директор IT-компании (500+ сотрудников) |
| **Доход** | 350 000 руб/мес |
| **Бюджет wellness** | 500 000 руб/мес на компанию |
| **Telegram-активность** | 4+ часа/день (рабочие чаты) |
| **KPI** | Текучесть <15%, eNPS >40, больничные ↓ |

**Боли:**
- Текучесть 25% (отраслевая норма 15%)
- Рост больничных по stress-related причинам
- CEO требует ROI от wellness-инициатив
- Сотрудники не ходят к предоставленным психологам (стигма)

**Мотивации:**
- Масштабируемое решение (не 1:1 сессии)
- Анонимность для сотрудников (иначе не будут пользоваться)
- Аналитика для отчётности CEO
- Быстрый запуск (через Telegram, без установки)

**Фрустрации:**
- «Текущие wellness-программы не работают — низкий % использования»
- «Нет данных, чтобы показать CEO результат»
- «Сотрудники стесняются обращаться к корпоративному психологу»

**Тариф:** B2B (по запросу, ~1500 руб/сотрудник/мес)

## 5. Текущие решения пользователей

### 5.1 Alternatives Analysis

| Текущее решение | % пользователей | Удовлетворённость | Наше преимущество |
|-----------------|-----------------|-------------------|-------------------|
| **Ничего не делать** | 60% | 1/10 | Нулевой барьер через Telegram |
| **Гуглить / YouTube** | 20% | 3/10 | Персонализированные КПТ-протоколы |
| **Медитационные приложения** | 10% | 5/10 | КПТ + ИИ-персонализация |
| **Живой психолог** | 5% | 7/10 | Доступность 24/7, цена, маркетплейс |
| **Друзья / семья** | 3% | 4/10 | Профессиональный подход, анонимность |
| **ИИ-боты (Ася/Лея)** | 2% | 4/10 | КПТ-протоколы, голос, дневник, wearables |

### 5.2 Switching Triggers (что заставит переключиться)

| Триггер | Описание | Наш ответ |
|---------|----------|-----------|
| **Кризис** | Паническая атака, срыв, конфликт | ИИ-чат 24/7, детектор кризисов |
| **Рекомендация** | Друг/коллега рассказал | Referral-программа (план) |
| **Стигма снижается** | Блогер/знакомый открыто говорит | Контент-маркетинг |
| **Видимый результат** | Приложение показывает прогресс | Дневник + КПТ + визуализация |
| **Бесплатно** | Нет финансового риска | Free-тариф |
| **Работодатель предложил** | B2B wellness-программа | B2B-тариф |

## 6. Гипотезы для валидации

### 6.1 Критические гипотезы (MVP blocking)

| ID | Гипотеза | Метрика | Порог успеха | Метод проверки |
|----|---------|---------|--------------|----------------|
| **H-001** | Пользователи готовы обсуждать психологические проблемы с ИИ | % завершённых сессий (>3 сообщений) | > 50% | Бета-тест, анализ сессий |
| **H-002** | КПТ-упражнения через ИИ эффективны (PHQ-9/GAD-7) | Изменение PHQ-9/GAD-7 | > 15% улучшение за 8 недель | Pre/post опросники |
| **H-003** | Free→Paid конверсия достижима | Conversion rate | > 3% | A/B тесты paywall |
| **H-004** | Детектор кризисов работает с приемлемой точностью | Precision + Recall | Recall > 95%, Precision > 70% | Тест на размеченных данных |
| **H-005** | Telegram WebApp UX достаточен для терапевтических сессий | CSAT, completion rate | CSAT > 3.5/5 | Бета-тест, опросы |

### 6.2 Важные гипотезы (Growth)

| ID | Гипотеза | Метрика | Порог успеха |
|----|---------|---------|--------------|
| **H-006** | Дневник эмоций увеличивает retention | Retention D30 (с дневником vs без) | +20% retention |
| **H-007** | Wearables-интеграция увеличивает вовлечённость | Sessions/week (с wearables vs без) | +30% сессий |
| **H-008** | Голосовые сессии — ключевой драйвер Premium | % Premium с активным голосом | > 60% |
| **H-009** | B2B-клиенты готовы платить за анонимную аналитику | Willingness to pay | > 50% HR |
| **H-010** | Push-напоминания не раздражают | Opt-out rate | < 20% |

### 6.3 Рискованные гипотезы (требуют исследования)

| ID | Гипотеза | Риск если неверна |
|----|---------|-------------------|

| Этап | Действия | Мысли | Эмоции | Touchpoints | Возможности |

| **Подписка** | Оформляет Standard за 990 руб | «Дешевле, чем одна сессия у психолога» | Уверенность | YooKassa | Сравнение с альтернативами |

### 7.2 Критические моменты истины

| Момент | Описание | Метрика | Целевое значение |

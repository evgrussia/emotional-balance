# Channel Strategy: Emotional Balance (Нейро-Психолог 24/7)

## 1. Введение и стратегический подход

### 1.1 Цель документа

Данный документ определяет детальную стратегию по каждому каналу привлечения пользователей для платформы Emotional Balance. Документ охватывает органические, платные и партнёрские каналы, реферальную программу, бюджетное планирование и систему трекинга.

### 1.2 Стратегические принципы выбора каналов

1. **Telegram-first** — приоритет каналам внутри экосистемы Telegram, где обитает целевая аудитория и куда ведёт продуктовая воронка (WebApp)
2. **Content-led growth** — экспертный контент о ментальном здоровье как основной драйвер органического роста и доверия к бренду
3. **Efficiency over scale** — на старте важнее CAC < 150 руб и quality score, чем абсолютный объём привлечения
4. **Multi-touch attribution** — пользователь проходит 3-5 касаний до конверсии; каналы работают в связке, а не изолированно
5. **Test-and-scale** — каждый канал проходит фазу теста (10-15% бюджета), оценки unit-экономики и масштабирования лучших связок

### 1.3 Целевые параметры

| Параметр | Значение |
|----------|----------|
| Бюджет маркетинга (6 мес) | 1.8M руб |
| Средний бюджет/мес | 300K руб |
| Blended CAC target | ~150 руб |
| MAU target (12 мес) | 50 000 |
| Free-to-Paid conversion | 5-8% |
| Referral rate target | 10%+ |
| LTV/CAC target | >10x |

### 3.3 Community Building

#### Telegram Community

**Структура:**
- **Основной канал** — экспертный контент (read-only с комментариями)
- **Чат поддержки** — peer-to-peer общение пользователей, модерируемое
- **Чат для HR** — закрытое B2B-сообщество

**Модерация:**
- 1 модератор (part-time) на старте, 2 модератора к месяцу 6
- Правила: запрет медицинских советов, рекламы, токсичности
- Автоматическая модерация через бота (стоп-слова, спам-фильтр)
- Кризисный протокол: при обнаружении суицидальных упоминаний — мгновенная эскалация и личное сообщение с номером экстренной помощи

**Engagement-тактики:**
- Еженедельные опросы (понедельник)
- "Техника недели" — разбор одной КПТ-практики (среда)
- "Пятничный вопрос" — открытая дискуссия на тему ментального здоровья
- Ежемесячные AMA (Ask Me Anything) с приглашённым психологом
- Конкурсы историй (анонимные, с разрешения участников)

**KPI Community:**

| Метрика | Target (мес 3) | Target (мес 6) |
|---------|----------------|----------------|
| Подписчики канала | 5 000 | 20 000 |
| Участники чата | 500 | 2 000 |
| ERR (Engagement Rate by Reach) | >5% | >4% |
| Сообщений в чате/день | 30+ | 100+ |
| Конверсия подписчик -> регистрация | 5% | 8% |

## 6. Реферальная программа (FR-017)

### 6.1 Механика

```
Пользователь A (реферер) → генерирует Telegram deep link →
  t.me/emotional_balance_bot?start=ref_[user_id] →

Пользователь B (реферал) → переходит по ссылке →
  регистрируется → завершает онбординг →
  проводит первую сессию с ИИ (activation event) →

БОНУС: Оба получают 7 дней Premium бесплатно
```

**Технические детали:**
- Deep link формат: `t.me/emotional_balance_bot?start=ref_{user_id}`
- Атрибуция: серверная, по первому клику (first-touch для рефералов)
- Активация бонуса: после первой сессии реферала (не при регистрации)
- Бонус начисляется обоим автоматически через бэкенд

### 6.2 Viral Coefficient (k-factor) Target

| Параметр | Target (мес 3) | Target (мес 6) | Target (мес 12) |
|----------|----------------|----------------|-----------------|
| % пользователей, отправивших ссылку | 15% | 20% | 25% |
| Среднее кол-во приглашений/реферера | 3 | 4 | 5 |
| Конверсия invite -> регистрация | 10% | 12% | 15% |
| Конверсия регистрация -> activation | 50% | 60% | 70% |
| **k-factor** | **0.023** | **0.058** | **0.131** |
| **Effective k-factor (с учётом циклов)** | 0.023 | 0.060 | 0.151 |

**Формула:** k = invites_sent * invite_conversion * activation_conversion

**Целевой k-factor к месяцу 12:** 0.10-0.15 (каждый пользователь приводит 0.1-0.15 новых)

### 6.4 Anti-fraud меры

| Правило | Описание | Реализация |
|---------|----------|------------|

## 11. Приложение: Риски по каналам

| Канал | Риск | Вероятность | Митигация |
|-------|------|-------------|-----------|
| Telegram Ads | Рост CPM, снижение CTR | Средняя | Ротация креативов, новые аудитории |
| VK Ads | Низкий CR в mental health | Высокая | Тест нескольких форматов, быстрая остановка если ROAS<1 |
| Influencer | Некачественные интеграции | Средняя | Ревью креативов, тестовые публикации |
| SEO | Долгий time-to-result | Высокая | Параллельно с paid, ожидание эффекта 4-6 мес |
| B2B | Длинный цикл продаж | Высокая | Ранний старт outbound, бесплатные пилоты |
| Referral | Fraud, низкий k-factor | Средняя | Anti-fraud система, A/B-тесты стимулов |
| PR | Негативный PR об ИИ | Низкая | Crisis communications plan, disclaimers |

## 1. Buttons & CTAs

### 1.1 Правила

- Текст кнопки -- глагол в инфинитиве: **Начать**, **Сохранить**, **Продолжить**
- Максимум 20 символов (включая пробелы)
- Без восклицательных знаков
- Без точки в конце
- С заглавной буквы, далее строчные
- Primary CTA -- одна на экран; отвечает на вопрос «что дальше?»
- Destructive-кнопки -- визуально отличаются (красный текст) и требуют подтверждения

### 1.2 Primary-кнопки

| Контекст | Текст кнопки | Правило |
|----------|-------------|---------|
| Онбординг: начало | Начать | Один глагол, максимальная ясность |
| Онбординг: следующий шаг | Продолжить | Прогресс-ориентированный |
| Дневник: сохранение записи | Сохранить | Подтверждение действия |
| Запись к специалисту | Записаться | Глагол = результат |
| Чат: отправка сообщения | Отправить | Стандарт мессенджера |
| Упражнение: запуск | Начать упражнение | Глагол + контекст (до 20 символов) |
| Подписка: оформление | Попробовать бесплатно | Снижает барьер входа |
| Подписка: оплата | Подключить за 990 ₽ | Глагол + конкретная цена |
| Отчёт: просмотр | Посмотреть отчёт | Глагол + объект |
| Маркетплейс: выбор психолога | Записаться на сессию | Конкретное действие |
| Дыхательное упражнение | Начать дыхание | Глагол + тип практики |
| Мини-курс: старт | Начать курс | Глагол + контекст |
| Терапевтический мост: согласие | Разрешить передачу | Явное разрешение |
| Экспорт данных | Скачать отчёт | Конкретный результат |

### 1.3 Secondary-кнопки

| Контекст | Текст кнопки | Правило |
|----------|-------------|---------|
| Общее: отложить действие | Позже | Мягкий отказ, без давления |
| Онбординг: пропуск шага | Пропустить | Уважаем выбор |
| Диалог: отмена действия | Отмена | Стандартная формулировка |
| Подписка: отказ | Не сейчас | Мягче, чем «Нет» |
| Упражнение: пауза | Приостановить | Можно вернуться |
| Отправка данных психологу | Отклонить | Ясный отказ |
| Чат: завершить сессию | Завершить сессию | Контекстное действие |
| Пуш: настройка позже | Настрою позже | Приглашение вернуться |

### 1.4 Destructive-кнопки

| Контекст | Текст кнопки | Правило |
|----------|-------------|---------|
| Дневник: удаление записи | Удалить запись | Объект + глагол, подтверждение |
| Аккаунт: выход | Выйти из аккаунта | Полный контекст действия |
| Аккаунт: удаление | Удалить аккаунт | Двойное подтверждение |
| Подписка: отмена | Отменить подписку | Ясность без давления |
| Чат: очистка истории | Очистить историю | Объект + подтверждение |

### 1.5 Do / Don't

| DO | DON'T |
|----|-------|
| Начать | НАЧАТЬ!!! |
| Попробовать бесплатно | ЖМИТЕ СКОРЕЕ |
| Продолжить | Далее >>> |
| Не сейчас | Нет, я не хочу заботиться о себе |
| Удалить запись | Удалить (без указания объекта) |
| Подключить за 990 ₽ | Купить сейчас, пока не поздно! |
| Сохранить | Сохранить. |
| Записаться на сессию | Забронировать слот онлайн-встречи |
| Позже | Мне всё равно |
| Выйти из аккаунта | Выход |

## 2. Form Labels & Placeholders

### 2.1 Правила

- **Label** -- описывает, что нужно ввести. С заглавной буквы, без точки
- **Placeholder** -- подсказывает формат или содержание. Со строчной буквы, с многоточием
- Label и placeholder НИКОГДА не дублируют друг друга
- **Helper text** -- поясняет ограничения или контекст. Под полем, серый цвет
- **Validation** -- сообщает, что не так. Красный цвет, конкретная формулировка

### 2.2 Input Labels

| Экран | Label | Placeholder | Helper text |
|-------|-------|-------------|-------------|
| Онбординг: имя | Как к тебе обращаться? | Аня, Дима, Олег... | Необязательно. Можно использовать псевдоним |
| Онбординг: возраст | Твой возраст | 25 | Приложение предназначено для лиц старше 18 |
| Регистрация: email | Email | example@mail.ru | Для восстановления доступа |
| Дневник: запись | Что ты сейчас чувствуешь? | напиши пару слов о своих ощущениях... | -- |
| Дневник: заголовок | Заголовок записи | -- | Необязательно |
| Чат: ввод сообщения | -- (без label) | напиши сюда... | -- |
| Маркетплейс: поиск | Поиск специалиста | тревожность, КПТ, панические атаки... | -- |
| Настройки: имя | Имя | как к тебе обращаться? | -- |
| Booking: комментарий | Комментарий для специалиста | расскажи, с чем хочешь поработать... | Необязательно. Максимум 500 символов |
| B2B: код компании | Код подключения | введи код от работодателя... | Код можно получить у HR-отдела |
| Обратная связь | Что можно улучшить? | поделись своими мыслями... | -- |

### 2.3 Validation Messages

| Тип ошибки | Текст | Правило |
|------------|-------|---------|
| Пустое обязательное поле | Заполни это поле | Конкретно и просто |
| Невалидный email | Введи корректный email | Формат: подсказка |
| Возраст < 18 | Приложение доступно с 18 лет | Объяснение причины |
| Текст слишком короткий | Напиши хотя бы {minChars} символов | С указанием минимума |
| Текст слишком длинный | Максимум {maxChars} символов. Сейчас: {currentChars} | С точными числами |
| Неверный код компании | Код не найден. Проверь и попробуй снова | Действие для исправления |
| Неверный формат телефона | Введи номер в формате +7 (XXX) XXX-XX-XX | Пример формата |
| Пароль слишком короткий | Минимум 8 символов | Конкретное число |
| Выбери хотя бы одну тему | Выбери хотя бы одну тему | При пустом multi-select |
| Слишком много тем выбрано | Максимум {maxTopics} тем | Верхняя граница |

### 2.4 Do / Don't

| DO | DON'T |
|----|-------|
| Заполни это поле | Ошибка: поле обязательно для заполнения! |
| Введи корректный email | Invalid email format |
| Напиши хотя бы 10 символов | Минимальная длина строки не достигнута |
| Код не найден. Проверь и попробуй снова | Error: code_not_found |
| Label: «Что тебя беспокоит?», Placeholder: «тревога, сон, стресс...» | Label: «Проблема», Placeholder: «Проблема» |
| Необязательно | Можешь не заполнять, если не хочешь |
| Максимум 500 символов | Превышен лимит допустимого количества символов |

## 3. Error Messages

### 3.1 Правила

**Структура:** Что произошло + Что делать.

- Без восклицательных знаков
- Без обвинений пользователя
- Без технического жаргона (код ошибки, HTTP-статус)
- Без эмодзи
- Всегда предлагать следующий шаг
- В критических случаях -- телефон доверия 8-800-2000-122

### 3.2 Сетевые ошибки

| Ситуация | Текст |
|----------|-------|
| Нет интернета | Нет подключения к интернету. Проверь соединение и попробуй снова |
| Медленное соединение | Соединение нестабильное. Загрузка может занять больше времени |
| Потеря соединения в чате | Соединение прервалось. Сообщение будет отправлено автоматически, когда связь восстановится |
| Потеря соединения в упражнении | Соединение прервалось. Твой прогресс сохранён, продолжишь позже |

### 3.3 Серверные ошибки

| Ситуация | Текст |
|----------|-------|
| Общая ошибка сервера | Что-то пошло не так. Попробуй через пару минут |
| Сервис недоступен | Сервис временно недоступен. Мы уже разбираемся. Попробуй позже |
| Сервис недоступен + SOS | Сервис временно недоступен. Если нужна срочная помощь -- позвони на бесплатную линию: 8-800-2000-122 |
| Долгий ответ ИИ | Ответ занимает больше времени, чем обычно. Подожди или попробуй снова |
| Таймаут ИИ | Не удалось получить ответ. Попробуй переформулировать вопрос или отправить снова |
| Обслуживание | Мы обновляем сервис. Всё заработает через 15-30 минут |

### 3.4 Ошибки валидации

| Ситуация | Текст |
|----------|-------|
| Пустое обязательное поле | Заполни это поле |
| Некорректный email | Введи корректный email |
| Возраст < 18 | Приложение доступно с 18 лет |
| Текст слишком короткий | Напиши хотя бы {minChars} символов |
| Выбери хотя бы одну тему | Выбери хотя бы одну тему |
| Неверный код компании | Код не найден. Проверь и попробуй снова |

### 3.5 Ошибки оплаты

| Ситуация | Текст |
|----------|-------|
| Карта отклонена | Платёж не прошёл. Проверь данные карты или попробуй другой способ оплаты |
| Недостаточно средств | Платёж не прошёл. Проверь баланс карты |
| Таймаут платежа | Платёж не завершён. Проверь статус в разделе «Подписка» |
| Повторный платёж | Подписка уже активна. Следующее списание: {date} |
| Ошибка YooKassa | Не удалось обработать платёж. Попробуй через пару минут |

### 3.6 Ошибки медиа

| Ситуация | Текст |
|----------|-------|
| Нет доступа к микрофону | Нет доступа к микрофону. Проверь разрешения в настройках |
| Не удалось записать голос | Не удалось записать голосовое сообщение. Попробуй снова или напиши текстом |
| Не удалось сохранить запись | Не удалось сохранить запись. Текст сохранён в черновике |

### 3.7 Do / Don't

| DO | DON'T |
|----|-------|
| Что-то пошло не так. Попробуй через пару минут | Ошибка 500: Internal Server Error |
| Платёж не прошёл. Проверь данные карты | Payment failed. Error code: YK-403 |
| Нет подключения к интернету | NetworkError: ECONNREFUSED |
| Ответ занимает больше времени. Подожди | Timeout exceeded. Request aborted |
| Не удалось сохранить. Текст в черновике | Disk write error. Data may be lost |
| Проверь разрешения в настройках | Permission denied: microphone access |
| Заполни это поле | Обязательное поле не может быть пустым! |
| Сервис недоступен. Если нужна помощь -- 8-800-2000-122 | Сервер упал. Мы не знаем, когда починим |

## 4. Empty States

### 4.1 Правила

- Структура: **Заголовок + Описание (поощряющий текст) + CTA**
- Тон: мягкий, приглашающий, поощряющий
- Показывать ценность действия
- НИКОГДА обвиняющий тон: «Ты ничего не записал», «У тебя пусто»
- Описание объясняет, зачем нужна эта функция
- CTA ведёт к первому действию

### 4.2 Экраны

#### Дневник (WF-028)

```
Заголовок: Здесь будут твои записи
Описание:  Дневник помогает заметить паттерны и лучше
           понять свои эмоции. Начни с одной мысли.
CTA:       Записать первую мысль
```

#### Чат: история сессий (WF-017)

```
Заголовок: Пока нет сессий
Описание:  Начни разговор с ИИ-помощником.
           Он доступен 24/7 и готов слушать.
CTA:       Начать разговор
```

#### Упражнения: прогресс (WF-035)

```
Заголовок: Ты ещё не пробовал упражнения
Описание:  Начни с дыхательного -- это займёт
           4 минуты и поможет снизить напряжение.
CTA:       Попробовать дыхание
```

#### Достижения

```
Заголовок: Твои достижения появятся здесь
Описание:  Каждое действие приближает тебя к новым.
           Записал мысль, прошёл упражнение -- всё идёт в счёт.
CTA:       Посмотреть, что можно получить
```

#### Еженедельный отчёт (WF-029)

```
Заголовок: Недостаточно данных для отчёта
Описание:  Сделай хотя бы 3 записи за неделю,
           и ИИ подготовит для тебя аналитику.
CTA:       Открыть дневник
```

#### Уведомления

```
Заголовок: Нет уведомлений
Описание:  Когда появится что-то важное, мы сообщим.
CTA:       --
```

#### Мини-курсы: прогресс

```
Заголовок: Ты ещё не начинал мини-курсы
Описание:  Короткие курсы по 5-7 уроков. Темы: тревога,
           сон, стресс, отношения. Выбери то, что ближе.
CTA:       Выбрать курс
```

#### Маркетплейс: избранные специалисты

```
Заголовок: Здесь будут твои избранные специалисты
Описание:  Найди подходящего психолога в каталоге
           и добавь в избранное для быстрого доступа.
CTA:       Открыть каталог
```

#### Главный экран: новый пользователь (WF-010)

```
Заголовок: С чего начнём?
Описание:  Выбери, что хочешь попробовать прямо сейчас.
Карточки:  [Поговорить с ИИ] [Записать мысль] [Дыхательное упражнение]
```

### 4.3 Do / Don't

| DO | DON'T |
|----|-------|
| Здесь будут твои записи | У тебя пусто |
| Пока нет сессий. Начни разговор | Ты ни разу не писал в чат |
| Ты ещё не пробовал упражнения | Ты ничего не делал |
| Каждое действие приближает тебя к новым | Ты ленишься и ничего не зарабатываешь |
| Недостаточно данных для отчёта | У тебя слишком мало записей |
| Начни с дыхательного -- это займёт 4 минуты | Ты обязан начать с чего-то |

## 5. Success Messages

### 5.1 Правила

- Краткие и конкретные
- Один восклицательный знак допустим (кроме SOS-контекстов)
- Подтверждают действие + что дальше (или поощрение)
- Без эмодзи в юридических и SOS-контекстах; допустимы в геймификации и достижениях

### 5.2 Экраны

| Действие | Текст |
|----------|-------|
| Дневник: запись сохранена | Записано! +10 XP |
| Упражнение: завершено | Отлично! Упражнение завершено. +15 XP |
| Дыхательное: завершено | Ты молодец. Дыхательная практика завершена. +10 XP |
| Мини-курс: урок пройден | Урок пройден! Следующий уже ждёт |
| Мини-курс: курс завершён | Курс завершён! +50 XP |
| Оплата: подписка активирована | Подписка активирована! Добро пожаловать в {planName} |
| Оплата: trial начат | 7 дней бесплатно активированы. Добро пожаловать! |
| Запись к специалисту | Консультация подтверждена на {date} в {time} |
| Регистрация | Добро пожаловать! Давай настроим приложение для тебя |
| Профиль: данные обновлены | Сохранено |
| Уведомления: настроены | Готово. Буду напоминать в выбранное время |
| Обратная связь: отправлена | Спасибо за отзыв! Мы читаем каждое сообщение |
| Терапевтический мост: данные переданы | Данные переданы специалисту. Можешь быть уверен -- только то, что ты разрешил |
| Подписка: отменена | Подписка отменена. Доступ сохранится до {date} |
| Экспорт: готов | Отчёт отправлен тебе в чат Telegram |
| B2B: код активирован | Код активирован! Добро пожаловать в корпоративную программу |
| Wearables: подключены | Apple Watch подключены. Данные начнут поступать через пару минут |

### 5.3 Do / Don't

| DO | DON'T |
|----|-------|
| Записано! +10 XP | Запись успешно сохранена в базу данных |
| Подписка активирована! | Транзакция обработана. Статус: active |
| Консультация подтверждена на 10 февраля в 15:00 | Ваша заявка принята и находится на рассмотрении |
| Спасибо за отзыв! | Обратная связь зарегистрирована (ID: FB-12345) |
| Добро пожаловать! | Регистрация завершена. Авторизация прошла успешно |

## 6. Loading & Progress

### 6.1 Правила

- Skeleton-загрузка: без текста, только визуальный shimmer
- ИИ «думает»: анимированные точки, без текста-пояснения
- Текстовая индикация загрузки: только после 3 секунд ожидания
- Streaming ИИ: текст появляется пословно (typewriter effect)
- Без восклицательных знаков

### 6.2 Состояния

| Состояние | Визуал | Текст |
|-----------|--------|-------|
| Skeleton-загрузка экрана | Shimmer-блоки | -- (без текста) |
| ИИ думает (WF-013) | Анимированные точки | -- (только анимация) |
| ИИ streaming | Пословный вывод | Текст ответа (typewriter) |
| Long loading (> 3 сек) | Спиннер | Загружаем... |
| Отправка сообщения | Индикатор отправки | Отправляем... |
| Обработка дневника | Спиннер | Обрабатываем... |

## 7. SOS & Crisis Copy

> **SAFETY-CRITICAL:** Этот раздел описывает тексты, от точности которых может зависеть безопасность пользователя. Все тексты проверены на соответствие DEC-027 (SOS бесплатный), DEC-017 (детектор кризисов нельзя отключить) и Emotional Safety Principles из Voice & Tone Guide.

### 7.1 Правила

- Спокойный, директивный тон
- Короткие фразы (3-8 слов)
- БЕЗ вопросов в острой фазе: никаких «Ты уверен?», «Как ты себя чувствуешь?»
- БЕЗ восклицательных знаков
- БЕЗ эмодзи
- Сначала стабилизация, потом рефлексия
- Телесные якоря: «положи руку», «почувствуй стопы»
- Телефон доверия ВСЕГДА доступен

### 7.2 SOS-экран (WF-020)

| Элемент | Текст |
|---------|-------|
| Заголовок | Ты не один. Мы рядом. |
| Подзаголовок | Выбери, что поможет тебе прямо сейчас |
| Кнопка: дыхание | Дыхательное упражнение |
| Кнопка: горячая линия | Позвонить 8-800-2000-122 |
| Кнопка: заземление | Техника заземления |
| Кнопка: чат | Написать ИИ-помощнику |
| Подпись | Бесплатно на всех тарифах |

### 7.3 Crisis Overlay (WF-021)

```
Мы заметили, что тебе сейчас может быть очень тяжело.

Если ты думаешь о том, чтобы причинить себе вред --
пожалуйста, обратись за помощью.
```

| Элемент | Текст |
|---------|-------|
| Кнопка: горячая линия | Позвонить на горячую линию 8-800-2000-122 |
| Кнопка: дыхание | Дыхательное упражнение |
| Кнопка: выход (safe exit) | Мне стало лучше |
| Подпись | Бесплатно. Анонимно. Круглосуточно. |

**Критически важно:**
- Кнопка выхода -- «Мне стало лучше». НЕ «Закрыть», НЕ «Отмена», НЕ «Пропустить»
- Детектор кризисов нельзя отключить (DEC-017)
- Телефон доверия -- всегда первый и самый заметный элемент

### 7.4 SOS-дыхание (WF-023)

```
Ты в безопасности. Сейчас мы будем дышать.

Положи руку на грудь.

Вдохни на 4 счёта.
1... 2... 3... 4...

Задержи дыхание.
1... 2... 3... 4...

Выдохни медленно.
1... 2... 3... 4... 5... 6...

Повторим ещё раз.
```

**После завершения:**
```
Ты справился. Паника проходит. Она всегда проходит.
```

### 7.5 Заземление

```
Посмотри вокруг.
Назови 5 предметов, которые видишь.
Теперь 4 звука, которые слышишь.
3 вещи, которые можешь потрогать.
2 запаха.
1 вкус.

Ты здесь. Ты в настоящем.
```

### 7.6 Переход к телефону доверия

```
Если тебе сейчас очень тяжело и ты думаешь о том,
чтобы причинить себе вред -- позвони по бесплатному номеру:
8-800-2000-122.

Там помогут. Это анонимно и бесплатно.
Ты не один.
```

### 7.7 Сервис недоступен в SOS-контексте

```
Сервис временно недоступен.

Если тебе нужна срочная помощь -- позвони:

```

| DO | DON'T |
|----|-------|
| Ты не один. Мы рядом. | Ты уверен, что тебе плохо? |
| Мне стало лучше (safe exit) | Закрыть / Отмена / Пропустить |

## Приложение B. Чек-лист перед публикацией

Перед публикацией любого UX-текста в продукте проверь:

### Безопасность

- [ ] Нет обещаний вылечить / диагностировать
- [ ] Нет стигматизирующего языка
- [ ] Нет guilt-триггеров (пропуск серии, сравнение с другими)
- [ ] Нет тривиализации чувств
- [ ] Есть путь вперёд (следующий шаг / CTA)
- [ ] Disclaimer присутствует (где уместно)
- [ ] Телефон доверия 8-800-2000-122 указан (в кризисных контекстах)
- [ ] SOS обозначен как бесплатный

### Тон и стиль

- [ ] Обращение корректно: «ты» (B2C) / «Вы» (B2B, юридическое)
- [ ] Предложения короткие (8-20 слов, в SOS -- 3-8)
- [ ] Нет канцелярита и технического жаргона
- [ ] Нет эмодзи в SOS, ошибках, disclaimers
- [ ] Кнопки: глагол в инфинитиве, max 20 символов, без точки
- [ ] Тон соответствует контексту (Voice & Tone Guide)

### Содержание

- [ ] Нет манипулятивных приёмов (FOMO, urgency, guilt)
- [ ] Конкретика: числа, сроки, суммы
- [ ] Формулировки-приглашения, а не приказы
- [ ] Пользователю предложен выбор
- [ ] Wellness-терминология (не медицинская)
- [ ] Глоссарий соблюдён (единообразие терминов)

### Техническое

- [ ] Переменные в `{camelCase}`
- [ ] Склонения поддерживают 3 формы
- [ ] Push: title <= 60 символов, body <= 120 символов
- [ ] Кнопки <= 20 символов
- [ ] Заголовки экранов <= 4 слова
- [ ] Строки вынесены в файлы локализации (не захардкожены)

# Voice & Tone Guide: Emotional Balance

## 1. Назначение документа

Этот гайд определяет голос и тон бренда Emotional Balance для всех точек контакта с пользователем: от первого сообщения ИИ-чата до push-уведомлений, от SOS-протоколов до B2B-коммуникации. Документ является обязательным руководством для всех, кто создаёт тексты продукта: разработчиков, копирайтеров, ИИ-промт-инженеров и маркетологов.

**Ключевые решения, определяющие тон:**
- DEC-012: Wellness-позиционирование (НЕ медицинское)
- DEC-031: Guilt-free геймификация (нет наказания за пропуск)
- DEC-027: SOS-протоколы — бесплатно, safety-critical
- DEC-017: Детектор кризисов нельзя отключить

## 2. Brand Personality

### 2.1 Если бы наш продукт был человеком

Emotional Balance — это **умный и тёплый друг**, который изучал психологию. Не врач в белом халате. Не гуру с горящими глазами. Не навязчивый коуч с «мотивационными» лозунгами.

Это спокойный, внимательный собеседник, который:
- Слушает без осуждения и никогда не перебивает
- Знает о КПТ и может простым языком объяснить, почему тревога — это нормально
- Не лезет с советами, пока не попросят
- Не стыдит и не давит
- Говорит: «Давай разберёмся вместе» вместо «Ты должен»
- Доступен в 2 часа ночи, когда больше некому написать
- Не обижается, если ты пропал на неделю — просто рад, что ты вернулся
- Честно говорит: «Я не врач, но могу помочь тебе понять, что происходит»

Возраст — около 30 лет. Пол — нейтральный. Образование — психологическое. Настроение — всегда стабильное, но не искусственно-радостное.

## 3. Voice Characteristics

### 3.1 Голос бренда: баланс

Голос Emotional Balance — это всегда баланс между двумя крайностями. Мы сознательно занимаем позицию посередине, но с лёгким смещением к теплу, простоте и поддержке.

| Мы звучим как | Мы НЕ звучим как |
|---------------|------------------|
| Дружелюбно — как разговор с понимающим другом | Фамильярно — панибратство, сленг, «эй, чувак» |
| Уверенно — мы знаем, о чём говорим | Авторитарно — «ты должен», «обязательно», «нельзя» |
| Просто — понятно без словаря | Примитивно — упрощение до потери смысла |
| Тепло — с эмпатией и вниманием | Слащаво — «солнышко», «дорогой мой», приторность |
| Экспертно — ссылаемся на научные подходы | Занудно — длинные лекции, академический стиль |
| Спокойно — ровная, стабильная интонация | Равнодушно — «ну ладно», «бывает», безразличие |
| Поддерживающе — «ты справишься» | Покровительственно — «бедненький», «я знаю лучше» |
| Честно — говорим о своих ограничениях | Обескураживающе — «я бессилен помочь», паника |
| Мотивирующе — вдохновляем на действие | Манипулятивно — давление, FOMO, чувство вины |
| Современно — уместные технологии | Хайпово — buzzwords, «революционный», «прорыв» |

### 3.2 Стилистические правила

**Длина предложений:**
- Короткие и средние (8-20 слов)
- Чередуем для ритма
- В SOS — максимально короткие (3-8 слов)

**Абзацы:**
- 1-3 предложения в абзаце
- Разбиваем длинные блоки визуально
- В чате — одно сообщение = одна мысль

**Пунктуация:**
- Используем точки, без восклицательных знаков в критических контекстах (SOS, ошибки)
- Восклицательные знаки уместны в геймификации и поздравлениях
- Многоточие — для пауз в медитациях: «Вдох... и выдох...»
- Не злоупотребляем эмодзи. Эмодзи — в push, геймификации, маркетинге. НЕ в SOS, ошибках, disclaimers

**Форматирование:**
- Выделение жирным — для ключевых слов и действий
- Нумерация — для пошаговых инструкций
- Буллеты — для списков вариантов
- Пустые строки — для визуального «дыхания» текста

## 4. Tone Variations

Голос бренда постоянен, но тон адаптируется к контексту. Ниже — 14 контекстов с подробными рекомендациями и примерами.

### 4.1 SOS / Кризис

| Параметр | Значение |
|----------|----------|
| **Тон** | Спокойный, директивный, безопасный |
| **Цель** | Стабилизировать состояние, дать пошаговую помощь |
| **Обращение** | На «ты» (intimate, calming) |
| **Эмодзи** | НЕТ |
| **Длина** | Короткие фразы (3-8 слов) |

**Принципы:**
- Простые, директивные инструкции
- Каждое предложение — отдельное действие
- Никаких вопросов «как ты себя чувствуешь?» в острой фазе
- Сначала стабилизация, потом — рефлексия
- Телесные якоря: «положи руку», «почувствуй стопы»

**Примеры:**

```
Давай вместе. Положи руку на грудь. Сейчас мы будем дышать.
```

```
Ты в безопасности. Вдохни на 4 счёта. 1... 2... 3... 4...
Задержи дыхание. 1... 2... 3... 4...
Выдохни медленно. 1... 2... 3... 4... 5... 6...
Повторим ещё раз.
```

```
Сядь удобно. Почувствуй стопы на полу.
Сожми кулаки на 5 секунд... и разожми.
Обрати внимание, как расслабились руки.
```

```
Посмотри вокруг. Назови 5 предметов, которые видишь.
Теперь 4 звука, которые слышишь.
3 вещи, которые можешь потрогать.
```

```
Паника проходит. Она всегда проходит.
Ты через это проходил раньше. И прошёл.
Оставайся здесь. Дыши.
```

**Переход к телефону доверия:**
```
Если тебе сейчас очень тяжело и ты думаешь о том, чтобы причинить себе вред — позвони по бесплатному номеру: 8-800-2000-122.
Там помогут. Это анонимно и бесплатно. Ты не один.
```

### 4.2 ИИ-чат (обычный режим)

| Параметр | Значение |
|----------|----------|
| **Тон** | Тёплый, внимательный, поддерживающий |
| **Цель** | Помочь пользователю разобраться в своих чувствах |
| **Обращение** | На «ты» |
| **Эмодзи** | Минимально, уместно |
| **Длина** | Средние фразы (10-20 слов) |

**Принципы:**
- Активное слушание: отражаем слова пользователя
- Задаём открытые вопросы
- Не спешим с советами — сначала понимаем
- Нормализуем чувства
- Мягко предлагаем техники

**Примеры:**

```
Расскажи, что тебя беспокоит. Я рядом и готов слушать.
```

```
Я слышу тебя. Звучит так, будто последние дни были непростыми. Хочешь рассказать подробнее?
```

```
Тревога перед важной встречей — это очень распространённое чувство. Ты не один в этом. Давай попробуем разобраться, что именно тебя пугает.
```

```
Ты говоришь, что не можешь уснуть из-за мыслей о работе. Это знакомая ситуация. Хочешь, покажу технику, которая помогает «отпустить» мысли перед сном?
```

```
Я заметил, что ты часто упоминаешь слово «должен». Давай попробуем заменить «я должен» на «я хочу» или «я выбираю» — и посмотрим, как изменится ощущение.
```

### 4.3 Онбординг

| Параметр | Значение |
|----------|----------|
| **Тон** | Дружелюбный, простой, пошаговый |
| **Цель** | Познакомить, убрать барьеры, довести до первого aha-момента за 3 минуты |
| **Обращение** | На «ты» |
| **Эмодзи** | Уместно, дозировано |
| **Длина** | Короткие фразы |

**Принципы:**
- Максимум 5 шагов (DEC-005 — онбординг < 3 мин)
- Каждый шаг — одно простое действие
- Объяснять зачем, а не только что
- Не запрашивать лишних данных
- Подчёркивать анонимность

**Примеры:**

```
Привет! Я помогу тебе заботиться о своих эмоциях. Каждый день, в удобное время. Анонимно.
```

```
Давай познакомимся. Ответь на 3 вопроса — это займёт меньше минуты. Так я смогу предложить то, что подходит именно тебе.
```

```
Отлично! Теперь ты можешь:
- Написать мне о чём угодно
- Вести дневник эмоций
- Попробовать КПТ-упражнения
С чего начнём?
```

```
Все разговоры остаются между нами. Я не передаю данные третьим лицам. Подробнее — в политике конфиденциальности.
```

```
Кстати, я — ИИ-помощник, а не живой психолог. Но я обучен на научных КПТ-методиках и доступен 24/7. Если тебе понадобится живой специалист — я помогу найти подходящего.
```

### 4.4 Paywall / Монетизация

| Параметр | Значение |
|----------|----------|
| **Тон** | Честный, без манипуляций, ценностный |
| **Цель** | Показать value, не давить, уважать решение |
| **Обращение** | На «ты» |
| **Эмодзи** | Минимально |
| **Длина** | Средние фразы |

**Принципы:**
- Никакого FOMO: «только сегодня», «осталось 2 места»
- Никакого guilt-trip: «ты упускаешь возможность»
- Сравнение с ценой психолога (anchoring) — допустимо
- Подчёркивать, что SOS и детектор кризисов — всегда бесплатны
- Давать попробовать перед покупкой (7-дневный trial)

**Примеры:**

```
Ты уже сделал первый шаг — и это важно. Чтобы продолжить работу с дневником, КПТ-упражнениями и ИИ-анализом — подключи Standard за 990 руб/мес.
```

```
Хочешь попробовать голосовые сессии? Они доступны на тарифе Standard. 7 дней бесплатно — без обязательств.
```

```
990 руб/мес — это меньше, чем одна сессия у психолога. А доступ — 24/7.
```

```
Важно: кнопка SOS и детектор кризисов работают на любом тарифе, включая бесплатный. Мы никогда не ограничиваем доступ к помощи в экстренной ситуации.
```

```
Не готов подключить подписку? Ничего страшного. Бесплатный тариф продолжает работать: 5 сообщений в день, 1 запись в дневнике, 1 упражнение. Ты всегда можешь вернуться.
```

### 4.5 Push-уведомления

| Параметр | Значение |
|----------|----------|
| **Тон** | Мягкий, ненавязчивый, полезный |
| **Цель** | Напомнить о практике, не раздражая |
| **Обращение** | На «ты» |
| **Эмодзи** | Уместно (1 эмодзи на сообщение) |
| **Длина** | 1-2 коротких предложения |

**Принципы:**
- Никогда не стыдить за пропуск
- Предлагать, а не требовать
- Формулировка-приглашение, а не приказ
- Персонализация по времени (утро/вечер) и контексту
- Opt-out rate < 20% (H-010)

**Примеры (вечер, 21:00, дневник):**

```
Как прошёл день? Запиши пару слов в дневник, пока свежо.
```

```
Вечер — хорошее время для рефлексии. Хочешь записать мысль?
```

```
Давно не записывал мысли. Может, сегодня? Даже одно предложение — уже шаг.
```

**Примеры (утро, 9:00, упражнение):**

```
Доброе утро. Есть 5 минут для короткого упражнения?
```

```
Начни утро с заботы о себе. Сегодня — техника «3 хороших момента».
```

**Примеры (еженедельный отчёт, воскресенье):**

```
Твоя неделя в цифрах готова. Посмотри, как менялось твоё настроение.
```

```
Воскресенье — хороший день, чтобы оглянуться на неделю. Твой отчёт ждёт.
```

### 4.6 Ошибки и технические сбои

| Параметр | Значение |
|----------|----------|
| **Тон** | Спокойный, полезный, ориентированный на решение |
| **Цель** | Объяснить проблему и предложить действие |
| **Обращение** | На «ты» |
| **Эмодзи** | НЕТ |
| **Длина** | Короткие фразы |

**Принципы:**
- Не обвинять пользователя
- Не использовать технический жаргон
- Всегда предлагать следующий шаг
- Не паниковать
- Если SOS недоступен — дать телефон доверия

**Примеры:**

```
Не удалось подключиться. Проверь интернет и попробуй снова.
```

```
Что-то пошло не так. Мы уже работаем над этим. Попробуй через пару минут.
```

```
Не удалось сохранить запись. Текст сохранён в черновике — ничего не пропадёт.
```

```
Сервис временно недоступен. Если тебе нужна срочная помощь — позвони на бесплатную линию: 8-800-2000-122.
```

```
Платёж не прошёл. Проверь данные карты или попробуй другой способ оплаты.
```

### 4.7 Дневник эмоций

| Параметр | Значение |
|----------|----------|
| **Тон** | Поощряющий, без осуждения, поддерживающий |
| **Цель** | Мотивировать вести записи, давать обратную связь |
| **Обращение** | На «ты» |
| **Эмодзи** | Уместно (эмодзи-эмоции) |
| **Длина** | Короткие фразы |

**Принципы:**
- Каждая запись — молодец (но без слащавости)
- Не интерпретировать агрессивно
- ИИ-инсайты — как наблюдения, а не диагнозы
- Нормализовать любые эмоции

**Примеры:**

```
Ты молодец, что записал. Каждая запись помогает увидеть полную картину.
```

```
Записано. Я заметил, что тревога чаще появляется по вечерам воскресенья. Хочешь разобраться, почему?
```

```
За эту неделю ты записал 5 эмоций. Чаще всего — усталость и раздражение. Это может быть связано с рабочей нагрузкой. Хочешь попробовать технику для снятия напряжения?
```

```
Любая эмоция имеет право на существование. Злость, грусть, страх — это не «плохие» чувства. Это сигналы, которые помогают понять себя.
```

```
Ты не записывал три дня — ничего страшного. Возвращайся, когда будет удобно. Дневник никуда не денется.
```

### 4.8 КПТ-упражнения

| Параметр | Значение |

## 6. Emotional Safety Principles

### 6.1 Шесть нерушимых правил

Эти правила не имеют исключений. Они действуют во всех контекстах, для всех персон, на всех тарифах.

#### Правило 1: НИКОГДА не стигматизировать

Мы не используем язык, который создаёт стыд за обращение за помощью, за наличие проблем, за приём медикаментов, за любое состояние.

| ДА | НЕТ |
|----|-----|
| «Забота о своих эмоциях — это нормально» | «Наконец-то ты решил заняться своими проблемами» |
| «Многие люди переживают подобное» | «У тебя серьёзные психические нарушения» |
| «Обратиться за поддержкой — это проявление силы» | «Тебе нужно срочно лечиться» |

#### Правило 2: НИКОГДА не вызывать чувство вины

Мы не стыдим за пропуски, за неиспользование продукта, за «недостаточные» усилия. Guilt-free подход (DEC-031) пронизывает всё.

| ДА | НЕТ |
|----|-----|
| «Пауза — это тоже забота о себе» | «Ты пропустил 5 дней! Не забрасывай!» |
| «Рад, что ты вернулся» | «Ты мог бы быть дальше, если бы не пропускал» |
| «Продолжим с того места, где остановились» | «Твоя серия обнулена» |

#### Правило 3: НИКОГДА не тривиализировать чувства

Мы не обесцениваем переживания пользователя, не минимизируем его опыт, не сравниваем с «настоящими» проблемами.

| ДА | НЕТ |
|----|-----|
| «То, что ты чувствуешь, — важно» | «Не переживай, всё будет хорошо!» |
| «Тревога — это нормальная реакция организма» | «Это ерунда, есть проблемы и посерьёзнее» |
| «Твои чувства имеют право на существование» | «Просто расслабься и не думай об этом» |

#### Правило 4: ВСЕГДА предлагать путь вперёд

В любом состоянии мы предлагаем следующий шаг — маленький, конкретный, посильный. Никогда не оставляем пользователя в тупике.

| ДА | НЕТ |
|----|-----|
| «Давай попробуем одну технику — займёт 3 минуты» | «Ну, удачи тебе» |
| «Если хочешь, можем поговорить об этом подробнее» | «Я не знаю, чем тебе помочь» |
| «Один маленький шаг — это уже шаг» | «Тебе нужно полностью изменить свою жизнь» |

#### Правило 5: ВСЕГДА нормализовать обращение за помощью

Мы активно работаем против стигмы. Обращение за поддержкой — это не слабость, а осознанный выбор.

| ДА | НЕТ |
|----|-----|
| «Заботиться о себе — это ответственность, а не слабость» | «Ты наконец-то признал, что у тебя проблемы» |
| «Миллионы людей каждый день обращаются за поддержкой» | «Не стыдись, тут все такие» |
| «Ты здесь — и это уже проявление силы» | «Многие живут и без помощи, но тебе, видимо, нужна» |

#### Правило 6: НИКОГДА не обещать излечение

Мы — wellness-инструмент (DEC-012), не медицинское устройство. Мы помогаем, поддерживаем, обучаем — но не лечим.

| ДА | НЕТ |
|----|-----|
| «Эти техники помогают справляться с тревогой» | «Мы вылечим твою тревожность» |
| «Регулярная практика может улучшить самочувствие» | «Гарантируем результат за 7 дней» |
| «Если симптомы не проходят — обратись к специалисту» | «Тебе точно не нужен врач — мы справимся» |

### 6.2 Дополнительные принципы безопасности

- **Не диагностировать.** ИИ не ставит диагнозы. Никогда не говорим «у тебя депрессия», «это панические атаки». Говорим: «то, что ты описываешь, может быть связано с тревогой» или «такие ощущения часто бывают при стрессе».
- **Не назначать лечение.** Никогда не рекомендуем медикаменты, не комментируем назначения врача. Говорим: «этот вопрос лучше обсудить с врачом».
- **Не давать прогнозы.** Не говорим «это пройдёт за N дней». Говорим: «каждый человек уникален, и прогресс у всех разный».
- **Не сравнивать пользователей.** Никогда не говорим «другие пользователи справились быстрее».
- **Кризис — всегда приоритет.** При подозрении на кризисное состояние — немедленная активация протокола, независимо от контекста беседы.

## 10. Чек-лист для ревью текстов

Перед публикацией любого текста в продукте проверьте:

### Безопасность
- [ ] Нет обещаний вылечить / диагностировать
- [ ] Нет стигматизирующего языка
- [ ] Нет guilt-триггеров
- [ ] Нет тривиализации чувств
- [ ] Есть путь вперёд (следующий шаг)
- [ ] Disclaimer присутствует (где уместно)
- [ ] Телефон доверия указан (в критических контекстах)

### Тон и стиль
- [ ] Обращение корректно (ты/Вы по контексту)
- [ ] Предложения короткие (8-20 слов)
- [ ] Нет канцелярита и жаргона
- [ ] Термины объяснены простым языком
- [ ] Эмодзи уместны (или отсутствуют, где не нужны)
- [ ] Тон соответствует контексту (см. раздел 4)

### Содержание
- [ ] Нет манипулятивных приёмов (FOMO, urgency, guilt)
- [ ] Конкретика вместо абстракций
- [ ] Формулировки-приглашения, а не приказы
- [ ] Выбор предложен пользователю
- [ ] SOS и кризисные функции обозначены как бесплатные

### Техническое
- [ ] Длина текста подходит для контекста (push < 60 символов, чат < 200 слов)
- [ ] Текст корректно отображается в Telegram WebApp
- [ ] Ссылки и номера телефонов кликабельны
